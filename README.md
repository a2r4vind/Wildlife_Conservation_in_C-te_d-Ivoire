# Wildlife_Conservation_in_C-te_d-Ivoire
Supervised learning project using deep learning (CNNs) to automatically identify and classify wildlife species from camera trap images in Côte d'Ivoire, supporting conservation efforts.

1. Project Title:  Wildlife Species Classification in Côte d'Ivoire
2. Project Description: "This project uses a dataset of camera trap images provided by the WQU Applied AI Lab course to build a classifier that helps conservationists monitor biodiversity and track endangered species in West Africa."
3. Goal/Objective: "To achieve accurate, automated classification of various species (e.g., hog, monkeys, antelopes) using supervised machine learning models."
